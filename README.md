## Nutshell
This project should allow us to accomplish the following:

  1) Install Vagrant
  2) Add the Trusty Image
  3) Install DevStack
  4) Install Magnum (Server and Client)
  5) Configure Magnum (Heat Kubernetes)
  6) Start Magnum
  7) Testing Magnum
